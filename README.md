# Diyet-Uygulamasi
Bu proje, bireylerin sağlıklı yaşam alışkanlıkları geliştirmelerine destek olmak amacıyla tasarlanmış bir mobil uygulamadır. Kullanıcılar, günlük olarak tükettikleri besinleri, içtikleri su miktarını ve vücut ölçülerini uygulama üzerinden kaydederek kendi gelişim süreçlerini takip edebilmektedir.
Uygulamanın öne çıkan özelliklerinden biri de kullanıcıların diyetisyenlerle birebir iletişim kurabilmesine olanak tanımasıdır. Bu sayede sadece bireysel takibe değil, aynı zamanda profesyonel desteğe dayalı bir sağlık yönetimi de mümkün hale gelmektedir.
